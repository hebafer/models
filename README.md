# SQLFlow Models

Premade Models for [SQLFlow](https://github.com/hebafer/sqlflow).

## Installation

This package is available on PyPI as `sqlflow_models`. So you can install it by running the following command:

  ```bash
  pip install sqlflow_models
  ```

## Development

## Prerequisite
### Python 3
`brew install python`

### Setup Environment
`make setup`

### Test
`make test`
